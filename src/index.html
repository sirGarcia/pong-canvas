Regras do Jogo Pong

  • O objetivo é mover sua raquete para rebater a bola e fazer o oponente errar.
  • Cada jogador controla uma raquete horizontal.
  • Use as teclas (a, d) para mover sua raquete para esquerda e direita.
  • Quando a bola passar pela raquete do oponente, você ganha um ponto.
  • O jogo continua até que um jogador alcance a pontuação máxima (a ser definida no seu código JS).
  • Divirta-se jogando Pong Multiplayer!
